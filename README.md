# ActivePath Physiotherapy — Demo Website

A CrateWeb sales demo for allied health prospects.

## Quick Start

```bash
npm install
npm run dev
```

## Build & Deploy (Cloudflare Pages)

```bash
npm run build
```

This produces a `dist/` folder ready for deployment.

### Cloudflare Pages Setup
1. Connect your Git repo or use direct upload
2. Build command: `npm run build`
3. Build output directory: `dist`
4. Node version: 18+

## Stack
- React 18 + Vite 6
- Tailwind CSS 3.4
- Single-page app with smooth scroll sections
